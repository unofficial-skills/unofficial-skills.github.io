YouTube alexa edition setup

andrews tech
So you want to have YouTube on alexa?

Video



If so lets get started. First off you need to setup a free Heroku account. You can do this by clicking here. Once we have created our Heroku account we need to deploy our code to it. Don't worry this is easy. Click the deploy to Heroku button.
Once on the page enter an app name. This can be anything you like, Once entered press deploy and wait for the app to install. After the app is installed click on the view button. This will take you to your app and will display a 404 error.

If you see this then your app is working. Take a note of your URL as you will need this later.

We now need an Api key from YouTube. We start by creating a project click here to create.
Choose a project name (this can be anything, I called mine potato-267419). Once created make sure you have your new project selected and head to the api console.

When your on the api console search for YouTube data api v3. Enable it and click the Create Credentials. Choose the following options stated in the image bellow.



Click the what credentials do I need button. It will then give you an api key, place this in your notes.


 Now we make the skill.  You now need a Voiceflow account. Click here to create one.

Now you have a Voiceflow account head to https://download.andrewstech.me/youtube-skill-v3

This should add the code to your account. As you may notice this is all drag and drop for this skill this
will help everyone to maintain the skill or add new functions. Please note that voiceflow is free and I highly
recommend them.
The two blocks that I have highlighted are where you api key and heroku link need to go. To edit them click on
their block and you should see the text "YOUR API HERE" replace this with the YouTube api key you were given.
In the next block you should see "YOUR LINK HERE" replace that with your heroku link.


Now you have done that Click on the settings cog at the top and change your Invocation name to that you want to use to launch the skill.
Once you have done that click upload to alexa and a button will appear saying connect amazon. CLICK IT.
and sign in to your amazon account. Once your account has been linked press the button again.

That is it the skill should now be available on your amazon echo.
